Introduction to HTML

What is HTML?

Hypertext Markup Language

Standardized system that tells the browser how to render (display) information by using tags and attributes as instructions.

Browsers transform tags into HTML elements that are contained in the Document Object Model, which is commonly referred to as the DOM.

CodePen

We will use CodePen as a tool for learning to write HTML. CodePen is a browser-based app that renders the HTML as we write it.
